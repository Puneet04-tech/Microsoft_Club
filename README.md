# Microsoft Club Website

    The Microsoft Club Website is the official website for the Microsoft Club at VIT Bhopal. It is designed to provide information about club events, teams, and contact details along with the club's mission and history.

    ## Table of Contents     - About the Project     - Features     - Project Structure     - Installation     - Usage     - Contributing     - License     - Contact

    ## About the Project

    This project features a modern, responsive design themed around the Microsoft Club at VIT Bhopal. The website includes several key pages:

    - Home Page (public/index.html): Welcomes visitors with a hero section, image carousel, and introduction.     - About Page (ublic/about.html): Details the club’s mission, history, and core values.     - Events Page (public/events.html): Lists upcoming events including workshops, bootcamps, talks, and interactive sessions.     - Teams Page (public/teams.html): Showcases different club teams and their roles.     - Contact Page (public/contact.html): Provides a form to get in touch with the club, plus additional contact information.

    ## Features

    - Responsive Design: Adapts to various screen sizes using CSS media queries.     - Interactive UI: Includes animations (such as form slide-up on submission and image carousel scrolling) for a dynamic user experience.     - Modern Styling: Uses gradients, shadows, and hover effects to create an engaging theme.     - Static Website: Uses plain HTML, CSS, and JavaScript making it easy to deploy and modify.

    ## Installation

    1. Clone the repository:

        sh         git clone <REPOSITORY_URL>         

    2. Navigate to the project directory:

        sh         cd microsoft-club-website         

    3. Install dependencies:

        sh         npm install         

    ## Usage

    To run the website locally, use the live server included in the dev dependencies:

    sh         npm start     

    This command launches live-server which serves the content from the public directory. Open your browser and navigate to the provided URL to view the site.
